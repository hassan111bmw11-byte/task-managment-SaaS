# TaskPro 🚀

TaskPro is a **modern Project Management Application** built with **Next.js (Full Stack)** and the **MERN stack**. It enables users to create projects and manage tasks within each project in a fast, scalable, and intuitive way.

---

## 📌 Overview

TaskPro helps users organize their workflow by structuring work into projects, where each project contains its own dedicated tasks.

Built with **Next.js** for performance, SEO optimization, and full-stack capabilities using API routes.

---

## ✨ Features

### 📁 Project Management
- Create, edit, and delete projects

- Organize work into multiple independent projects
- Clean and intuitive dashboard

### ✅ Task Management (Inside Projects)
- Open any project to view its tasks by click on the project's card
- Add tasks to specific projects
- Edit and delete tasks
- Update task status (To Do / In Progress / Done)
- Fully isolated tasks per project

### ⚡ UI & Experience
- Modern dashboard UI
- Fast navigation powered by Next.js routing
- Responsive design for all devices
- Smooth and clean user experience
